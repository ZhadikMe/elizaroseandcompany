Eliza Rose & Σία...
Λοιπόν, προς το παρόν είμαι μόνο εγώ! :)
Ωστόσο, έχω μεγάλες ελπίδες ότι μια μέρα τα δύο πολύτιμα δώρα μου από τον ουρανό θα μοιραστούν το πάθος μου για τη δημιουργία τέχνης. Το όνομα Eliza Rose είναι μια συνεργασία των δεύτερων ονομάτων των κορών μου. Η μία είναι λίγο γλυκιά, η άλλη λίγο νευρική, εξ ου και η έμπνευσή μου για τη δική σας προσωπική δημιουργία! Είναι τιμή μου που μπορώ να δημιουργήσω ένα σχέδιο μόνο για εσάς και το πολύτιμο σας αγγελούδι. Είτε το στυλ σας είναι κομψό, vintage, γεμάτο χρώμα ή (γλυκό ΚΑΙ νευρικό στην περίπτωσή μου), μαζί θα δημιουργήσουμε έναν εκπληκτικό καμβά για το σπίτι σας.
